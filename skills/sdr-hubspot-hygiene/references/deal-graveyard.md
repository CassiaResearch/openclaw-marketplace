# Deal Graveyard Process

_Canonical source: Sales Ops | Data Hygiene Handbook (Confluence page 2314928144, "Deal Graveyard" section)_
_Fetch from Confluence for the latest version. This file summarizes the key rules._

## What Is It?
Active deals with no external activity for 60+ days fall into the Deal Graveyard. Any rep can claim them.

## How to Claim a Deal

1. **Find deals:** Use the SDR Deal View in HubSpot to see graveyard-eligible deals.
2. **Verify:** Check the deal for missed communications. If a piece was missed, you may have to return the deal to the original rep.
3. **Reach out:** Call FIRST, then email with a personalized activated sequence (tailor the first email).
4. **Take over:**
   - (Contact page) Change contact owner & SDR owner to yourself
   - (Deal page) SDRs: move Demo Booked → Demo Incomplete. Deals already in Demo Incomplete stay there.
   - Create a new deal when a new meeting is booked (ensures clean historical trail)

## Key Rule
The process is to close the current deal and create a new one — never reuse the old deal. This ensures a clear data trail.
