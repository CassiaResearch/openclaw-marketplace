# emma-playbooks

Plugin library of skills/playbooks for Emma to use and refer to to do her work as an SDR.

## Skills

| Skill | Purpose |
|---|---|
| `sdr-cold-call` | End-to-end outbound cold call flow (call → qualify → CRM → handoff → prep) |
| `sdr-cold-email` | 7-email cold prospecting sequence, reply handling, demo booking |
| `sdr-hubspot-hygiene` | Post-call CRM hygiene, meeting outcomes, deal stage updates |
| `sdr-like-audience-list` | Build lookalike prospect lists from an AE's closed-won profile |
| `sdr-reactivation-list` | Prioritized reactivation list for an AE from stale/lost deals |
| `team-meet-and-greet` | Structured teammate intros via Slack DM (or voice if requested) |
